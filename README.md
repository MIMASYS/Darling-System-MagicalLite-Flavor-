## Descripcion

Darling System Lite es la version **monolitica clasica** del kit de herramientas para Windows 11/10. Todo el codigo en un solo archivo .ps1 para maxima portabilidad.

### Caracteristicas principales:

- Utilidades del sistema (CPU, RAM, discos, red)
- Herramientas de gestion (Taskmgr, servicios, Explorer)
- Mantenimiento (SFC, DISM, limpieza, punto de restauracion)
- Gestion de discos (SMART, formateo, reparacion USB)
- Opciones de arranque (BIOS, Modo Seguro)
- Optimizacion de Windows 11 (Copilot, Widgets, bloatware)
- Descargador de herramientas especializadas
- Perifericos y hardware (USB, impresoras, Bluetooth, camaras)
- Reparacion avanzada de red
- Modo tecnico (msinfo32, regedit, gpedit, etc.)
- Instalacion con Winget
- Reportes HTML y TXT
- Autoverificacion de actualizaciones
